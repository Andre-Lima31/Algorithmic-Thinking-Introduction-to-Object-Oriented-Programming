# Algorithmic-Thinking-Introduction-to-Object-Oriented-Programming
Projeto Python feito para trabalho do segundo semestre de CDC, UniFECAF, matéria de Algorithmic Thinking, aplicativo de calculo de aluguel baseado nas instruções dadas em aula, + geração de planilha .csv contendo o valor total final dos primeiros 12 meses
# Projeto feito com na linguagem Python 3.14.2, pelo programa de edição VSCode x64 1.70.2, importando bibliotecas csv e datetime
# Esse é um projeto simplista feito com objetivos educacionais, pela tentativa e erro, e auxilio de plataformas como w3school.com
# A versão 2.5 é a ultima e funcional versão do programa, as demais podem variar em termos de qualidade e segurança, umas funcionam mas apresentam erros ou brechas, outras funcionam mas estão incompletas, outras simplesmente não funcionam mas foram salvas por questão de manter preservado esses erros para evita-los no futuro.
# A pasta também inclui outros documentos .py que foram usados afim de testes separados, sem estragar o codigo inteiro ou haver a necessidade de passar pelo codigo inteiro até chegar a parte aonde o teste é desejado, porém foram salvos para lembrar de como certos codigos funcionam ou por questões de manter historico, e foram incluidos em uma pasta aparte.
